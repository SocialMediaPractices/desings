# Grupo Lormig — Design System README

## Overview

Grupo Lormig is building a unified design system intended to support branded interfaces, AI products, automation dashboards, websites, slide decks, internal tools, and future product surfaces under a consistent visual and interaction language.

The current direction emphasizes:

* Warm earthy visual identity
* Rounded and approachable interfaces
* Professional but human-centered interaction patterns
* Modular UI systems for AI-first workflows
* Reusable React-based component architecture
* Cross-product consistency for automation, CRM, operational dashboards, and future SaaS products

The design system is intended to function both as:

1. A visual brand foundation
2. A production-ready UI reference system for AI-assisted product generation

The project appears positioned for:

* AI automation products
* CRM workflows
* Agentic AI systems
* Operations dashboards
* Internal business tooling
* Customer-facing web experiences
* Presentation systems
* Prototype generation

# VISUAL FOUNDATIONS

## Color Direction

Core visual identity:

* Warm earthy palettes
* Clay tones
* Sand neutrals
* Warm browns
* Terracotta accents
* Deep charcoal grounding tones
* Muted natural greens
* Warm cream backgrounds

The system should avoid:

* Bright neon palettes
* Excessive pure blue gradients
* Cold corporate greys
* Oversaturated SaaS palettes

---

## Typography

Typography direction should prioritize:

* Readability
* Human warmth
* Professional structure
* Dashboard clarity

Recommended structure:

* Clean sans-serif primary UI font
* Optional serif display accent for brand moments
* Strong hierarchy system
* Generous spacing

Potential font direction:

* Inter
* Manrope
* Plus Jakarta Sans
* General Sans
* Satoshi

Potential serif companion:

* Lora
* Cormorant Garamond
* Instrument Serif

---

## Corner Radius

Rounded corners are a major visual characteristic.

Expected direction:

* Large radii
* Soft containers
* Friendly card structures
* Minimal sharp geometry

Likely scale:

* Buttons: 12–18px
* Cards: 20–28px
* Modals: 24–32px

---

## Shadows & Elevation

Preferred:

* Soft shadows
* Low contrast depth
* Diffused elevation
* Warm shadow tinting

Avoid:

* Hard black shadows
* Heavy neumorphism
* Sharp floating effects

---

## Layout Philosophy

Preferred layouts:

* Spacious
* Grid-based
* Structured
* Calm density
* High readability

Likely characteristics:

* Strong whitespace
* Clear sections
* Stable navigation
* Dashboard modularity
* Large cards
* Wide panels

---

## Motion & Animation

Recommended motion direction:

* Subtle fades
* Soft easing
* Gentle scaling
* Calm transitions

Avoid:

* Aggressive bounce animations
* Overly elastic movement
* Excessive motion-heavy interactions

---

## Hover States

Preferred:

* Slight elevation
* Soft darkening/lightening
* Opacity adjustments
* Warm highlight glow

---

## Press States

Recommended:

* Slight scale reduction
* Deeper shadow compression
* Slight darkening

---

## Transparency & Blur

Should be used sparingly.

Good use cases:

* Floating navigation
* Modals
* AI overlays
* Context panels

Avoid:

* Excessive glassmorphism
* Over-blurred interfaces
